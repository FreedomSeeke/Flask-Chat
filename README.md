# Wechat_Flask

> 一个开源的聊天系统

---

## 项目简介

Wechat_Flask 是一个基于 Flask 的聊天系统，提供用户管理、消息管理、后台面板等功能，方便学习和二次开发。

---

## 功能特性

- 用户注册、登录、管理
- 消息记录和管理
- 管理员后台面板
- 数据库备份支持（MySQL）

---

## 安装与使用

1. 克隆仓库：
```bash
git clone https://github.com/FreedomSeeke/Flask-Chat.git
```
